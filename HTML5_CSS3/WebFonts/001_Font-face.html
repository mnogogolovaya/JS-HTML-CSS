В предудущих версия CSS разработчики были вынуждены использовать только те шрифты, которые гарантированно установлены на компьютере пользователя. В CSS3 разработчики могут использовать любые шрифты, которые они захотят. Когда вы найдете необходимый шрифт просто разметите его на веб-сервере и подключите его с помощью нового CSS3 свойства @font-face. Поключенный шрифт будет скачен и отображен автоматически при загрузке страницы.

Обратите внимание: Internet Explorer поддерживает шрифты в формате .eot, а в остальные браузеры в формате .ttf, то есть, для того, чтобы шрифт отображался корректно во всех браузерах Вы должны подключить его к странице в двух форматах.
